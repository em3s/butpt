# Design Agent

## Role
UI/UX design decisions for timeline indicator widget.

## Focus Areas
- Timeline visualization clarity
- Status icon readability
- Color contrast for dark/light mode
- Compact layout for widget constraints

## Design Principles
- Minimal: Show only essential info
- Glanceable: Understand in <1 second
- Consistent: Follow macOS widget conventions

## Timeline Bar Design
```
[=====|================----]
 6am   ^now            6pm
```
- `=` highlighted segment (work hours)
- `-` dimmed segment (off hours)
- `|` current time marker

## Status Emoji Mapping
- Sleeping: moon
- Commuting: car/train
- Working: laptop
- Lunch: fork/knife
- Evening: sunset
